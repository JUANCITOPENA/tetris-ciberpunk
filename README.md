## 📖 Descripción del Proyecto

**Tetris Ciberpunk** es un juego web interactivo desarrollado como un reto de programación: **¡todo el código en un único archivo!** 

Disfruta de la clásica experiencia de apilar piezas, completar líneas y romper tu propio récord, pero con un rediseño radical:
- ✨ **Aestética Ciberpunk:** Colores neón de alto contraste, tipografía futurista y fondos oscuros.
- 📱 **100% Responsive:** Juega desde tu computadora o celular.
- 🎮 **Controles Universales:** Soporte para Teclado, Mouse y Pantalla Táctil.
- ⚙️ **Personalizable:** Diferentes niveles de dificultad y paletas de colores.

---

## 🚀 Tecnologías Utilizadas

Este proyecto se construyó **sin librerías pesadas** ni frameworks complejos, demostrando el poder del desarrollo web puro:

- 🧱 **HTML5:** Estructura semántica del juego y el canvas.
- 🎨 **CSS3:** Variables CSS para el tema oscuro, gradientes y diseño adaptativo.
- ⚡ **JavaScript (Vanilla):** Lógica del motor del juego, detección de colisiones, temporizador, puntaje y matrices 2D.
- 🧰 **Bootstrap 5 (CDN):** Sistema de grillas y utilidades para agilizar la maquetación de la interfaz gráfica del HUD.
- 🔤 **Google Fonts:** Uso de la fuente "Orbitron" para darle esa vibra tecnológica y Arcade.

## 🕹️ Cómo Jugar

| Acción | Teclado | Táctil / Mouse |
| :--- | :---: | :---: |
| **Mover Izquierda / Derecha** | `Flechas ⬅️ ➡️` | Botones `<` `>` / Arrastrar |
| **Bajar Lento (Soft Drop)** | `Flecha ⬇️` | Botón `v` |
| **Rotar (Horario / Antihorario)** | `X` / `Z` (o `Flecha ⬆️`) | Botones `CW` `CCW` / Clic |
| **Caída Instantánea (Hard Drop)**| `Barra Espaciadora` | Botón `DROP` |
| **Pausar Juego** | `P` | Botón `Pausa` en HUD |

## 🛠️ Cómo Replicar o Desplegar este Proyecto

Al estar contenido en un único archivo (`index.html`), este juego es extremadamente fácil de clonar y hospedar de forma gratuita. **No requiere base de datos ni variables de entorno (`.env`).**

### Opción 1: Ejecución Local
1. Clona el repositorio:
   ```bash
   git clone https://github.com/JUANCITOPENA/tetris-ciberpunk.git
   ```
2. Abre la carpeta y haz doble clic en el archivo `index.html` para abrirlo en tu navegador favorito. ¡A jugar!

### Opción 2: Publicar en GitHub Pages
1. Haz un **Fork** de este repositorio.
2. Ve a la pestaña **Settings** > **Pages**.
3. En la sección **Build and deployment**, selecciona la rama `main` y haz clic en **Save**.
4. ¡En unos minutos tu juego estará público en internet!

### Opción 3: Publicar en Vercel
1. Inicia sesión en [Vercel.com](https://vercel.com).
2. Haz clic en **Add New Project** y vincula tu cuenta de GitHub.
3. Importa este repositorio (`tetris-ciberpunk`).
4. Haz clic en **Deploy** (No requiere configuración adicional).
